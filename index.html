Quiz Game Features

This quiz game consists of 10 questions which must be answered within 5 minutes from starting the quiz.

Each question has 4 answer options from which users can select one of the option by clicking on the button preceeding the answer.

Users must attempt to answer all questions before submiting the quiz.

To submit your answers for evaluation, click "Submit" and your score will be displayed.
